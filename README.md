## Why This Matters

Game reviews can vary significantly depending on scoring systems, editorial policies, audience demographics, and platform preferences. A single score rarely tells the full story.

This dashboard promotes transparency by making differences measurable and visible, helping users make informed decisions based on aggregated data rather than isolated opinions.
